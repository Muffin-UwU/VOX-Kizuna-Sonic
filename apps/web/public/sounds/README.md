# Sound Files

This directory should contain the following pre-generated sound files:

- `stop.mp3`: High tone / "Tomare"
- `come.mp3`: "Oide" voice
- `danger.mp3`: Bell / "Kiken"

These files are expected to be generated using MiniMax TTS or similar tools before deployment.

If these files are missing, the application will fallback to synthesized beeps (Oscillators) for development purposes.
